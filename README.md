1. 数据量小(0~20)时：
    * 常见搜索算法DFS,BFS；这些算法的复杂度是O(n!),O(2^N)
2. 数据量大(20~10^6)时：
    * Tree Traversal, graph Traversal;
    * Heap Based;
    * sorting based;
    * 常见DP算法(包括linear DP & other DPs), greedy policy, divide & conquer policy
3. 数据量极大时：
    * Binary Search
    * Math answer
